About GeoScore

An open-source GEO auditing tool that synthesizes frameworks from the community's leading projects.

Scoring Methodology

GeoScore evaluates websites across 12 dimensions for Generative Engine Optimization (GEO) readiness, producing a 0-100 score. The scoring system synthesizes frameworks from 5 leading open-source projects.

Technical Notes

Data source: All analysis is performed client-side by fetching publicly accessible resources (robots.txt, llms.txt, HTML, JSON-LD). No user data is collected or stored on servers.
CORS handling: Some resources are fetched via a Cloudflare Worker proxy when direct browser access is blocked.
Privacy: Audit results are stored only in your browser's localStorage. No data is transmitted to third parties.
Limitations: JavaScript-rendered content may not be fully captured. For SPAs, consider static rendering or pre-rendering for best results.
